Classpublic class SWFByteArray
InheritanceSWFByteArray Object

	 Adds the ability to read and write core SWF data types to standard ByteArray.
	 Abbreviations from the SWF spec:
	 	SI = Signed Integer
	 	UI = Unsigned Integer
	 	SB = Snsigned Bits
	 	UB = Unsigned Bits
