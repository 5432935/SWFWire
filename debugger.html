SWFWire Debugger

An open source AIR application for intense ActionScript 3 debugging, similar to Firebug.

Implemented Features:

  • Automatic injection of logging statements into every method, allowing you to see the flow of the code, and benchmark each call.
  • Inspect display objects, by rolling over them and seeing their public properties.
  • Pass custom flashvars.
  • Logging of URLLoader and NetConnection network requests.

In Progress Features:

  • Inspection of private/protected properties.

Planned Features:

  • UI for tracking URLLoader and NetConnection network requests.
  • Memory profiling.
