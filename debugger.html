SWFWire Debugger

An open source AIR application for intense ActionScript 3 debugging, similar to Firebug.

Implemented Features:

  • Automatic injection of logging statements into every method, allowing you to see the flow of the code, and benchmark each call.
  • Inspect display objects, by rolling over them and seeing their properties.
  • Inspection of private/protected properties.
  • Pass custom flashvars.
  • UI for tracking URLLoader network requests.
  • Object list monitor and garbage collector.

In Progress Features:

  • Display list browser.
  • UI for tracking NetConnection, Socket and ServerSocket network requests.

Planned Features:

  • Memory profiling.
  • FPS profiling.
  • CPU profiling.
